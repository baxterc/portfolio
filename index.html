My Projects

I've made a few projects to gain proficiency in using HTML, CSS and Bootstrap. Take a look!

  • Wikipedia

    Uses HTML, CSS and Bootstrap. This was an attempt to reproduce Wikipedia's styling and formatting (rather than its behavior). The main goals here were to:

    1. Become comfortable using divs alongside Bootstrap's grid system to create columnar content.
    2. Use spans for inline formatting.
    3. Float images correctly based on the actual site's formatting.
  • Animal Shelter

    Uses HTML and CSS. This page primarily demonstrates some of the basic principles that make CSS powerful:

    1. Cascading - The order that CSS uses to determine how to combine rules.
    2. Inheritance - Rules applied to a parent element will apply to child elements nested within that parent element.
    3. Specificity - The most specific rule applied to an element takes precedence over other rules.
  • KISS Fan Site

    Uses HTML and CSS. A basic page designed to show how the box model works. Using CSS classes, block elements can be reshaped and realigned relative to one another.

  • Boring Lecture

    Uses HTML and CSS. This page uses CSS classes via spans and divs to create different text formats rather than using inline formatting via HTML.

  • Hello World

    The first page I made at Epicodus! It uses only HTML. The page itself introduces the most basic HTML elements. I also used it to get acquainted with how to use Git (making commits, pushing, cloning, and so on.)
